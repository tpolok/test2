Les avantages d'une PWA

Il y a énormément d'avantages à construire une PWA.

Temps de développement réduit

Le temps de développement peut être par exemple considérablement réduit si le projet s'y prête

En construisant une PWA, si le projet comporte la demande de création d'un site web ainsi qu'une application mobile, selons si le projet est compatible avec les désavantages d'une PWA (voir ici)

Alors la PWA est une bonne alternative !

En effet, il suffit d'avoir des connaissance en HTML/CSS et en Angular pour pouvoir réaliser ces deux composants !

Le temps de développement ne sera pas divisé par deux puisque la réalisation d'une PWA demande d'avoir un site responsive de façon parfaite ce qui peut prendre beaucoup de temps

Cependant, le gain de temps sera tout de même considérablement réduit surtout si la demande d'application mobile concernait Android et iOS

La PWA permettant cette formule 3 en 1, site web mais aussi application Android et iOS

Pas de store

Un autre avantage d'avoir une PWA est cette liberté de ne pas passer par les store mobile

Bien que le store Android soit largement abordable, le store iOS peut l'est moins

Ce qui peut être un frein dans certains cas

Ici, seul le coût du serveur est présent.

Un coup de boost

En effet, même si cet avantage n'est pas toujours présent, de nombreux sites / applications ont eu un boost de visite depuis qu'ils ont lancé leur PWA

Il existe même un site référençant les success story de PWA, elles sont nombreuses et croient d'années en années

Ce site est une compilation de faits sur les PWA, allant de site internet ayant doublé leur fréquentation aux gains de performances octroyé par la construction d'une PWA par rapport à l'application mobile

Il existe même un site référençant les PWA disponibles à l'heure actuelle !

Un gain de performance

Grâce au Service Worker permettant de mettre en cache votre site web ( cf les promesses d'une PWA )

Votre site web gagnera forcément en rapidité d'affichage !
